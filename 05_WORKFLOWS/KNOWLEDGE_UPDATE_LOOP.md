# Knowledge Update Loop

## 目标

外部资料抓取不是为了文件归档，而是为了更新 AI 认知。

## 流程

```text
collect raw sources
→ classify source quality
→ create reading cards
→ synthesize answer
→ update workflow/spec/skill
→ write stable conclusion
```

## 产物

- raw/
- clean/
- reading/
- insights/
- kb/STABLE_CONCLUSION.md
- workflow patch
