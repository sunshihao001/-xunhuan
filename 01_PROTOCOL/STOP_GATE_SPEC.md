# STOP_GATE_SPEC

## Done

满足全部验收标准，并且 verifier 证据完整。

## DoneWithRisk

核心目标达成，但存在非阻塞风险，必须明确写出：

- 风险是什么
- 影响范围
- 为什么不阻塞
- 需要人后续判断什么

## Blocked

必须停止的阻塞：

- 缺权限
- 缺信息
- 环境不可用
- 同一问题验证失败两次
- 需要安装依赖或联网但不在边界内
- Agent 输出无法被验证

## HumanGate

必须交还给人的场景：

- 产品方向变化
- 技术栈替换
- 密钥/账号/生产数据
- 不可逆 Git 操作
- 数据库迁移
- 部署
- 费用/预算风险
- 法务/合规/内容风险
