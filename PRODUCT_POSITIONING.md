# Product Positioning / 产品定位

## 产品名

Xunhuan Agent OS / 循环代理操作系统

## 定位

一个面向个人和项目的 AI 工作流操作系统，用 Loop Engineering 把 AI 从“聊天助手”升级为“可控循环执行系统”。

## 不是

- 不是 prompt 模板集合
- 不是单个 Agent
- 不是简单自动化脚本
- 不是无界自治系统

## 是

- 目标编译器
- 有界 loop 协议
- 单轮 work order 生成器
- Agent runner 接入层
- Verifier gate
- State / handoff 管理器
- Knowledge patch 系统

## 核心用户价值

让用户不再成为每轮 agent 执行中的瓶颈。

```text
用户定义目标和边界
系统编译 loop
Agent 执行
Verifier 检查
系统决定继续/停止/交还
```
